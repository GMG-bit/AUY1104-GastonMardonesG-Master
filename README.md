## Repositorio que contiene el Workflow Reutilizable

### Introducción: ¿Qué son los Reusable Workflows?
Los Reusable Workflows (workflows reutilizables) son una funcionalidad de GitHub Actions que permite centralizar la lógica de CI/CD en un repositorio único y
consumirla desde múltiples proyectos, evitando la duplicación de código en pipelines.
En lugar de copiar y pegar el mismo bloque YAML en cada repositorio, defines el workflow una sola vez y lo referencias desde cualquier otro repositorio con una
línea de código.
